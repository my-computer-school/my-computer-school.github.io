1. 메모 앱 개요

메모 앱은 사용자가 개인 메모를 작성, 저장, 관리할 수 있는 간단하고 안전한 노트 작성 도구입니다.

📝 주요 기능
  • 텍스트 메모 작성 및 편집
  • 메모 카테고리 분류
  • 검색 및 필터링
  • 메모 내보내기
🛡️ 개인정보 보호 원칙
  • 최소한의 정보만 수집
  • 로컬 저장 우선
  • 암호화된 데이터 전송
  • 투명한 정보 제공

2. 수집하는 정보

중요: 메모 앱은 사용자의 개인정보를 최소한으로 수집합니다.
✅ 수집하는 정보
  • 메모 내용: 사용자가 작성한 텍스트
  • 메타데이터: 작성/수정 날짜, 카테고리
  • 기술적 정보: 앱 버전, 오류 로그 (개인식별 불가)
❌ 수집하지 않는 정보
  • 이름, 이메일 등 개인식별정보
  • 위치 정보
  • 연락처 정보
  • 기기 고유 식별자

3. 정보 이용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

수집 정보 이용 목적 보관 위치
메모 내용 메모 저장, 검색, 편집 기능 제공 사용자 기기 로컬 저장소
메타데이터 메모 정렬, 분류, 관리 기능 사용자 기기 로컬 저장소
기술적 정보 앱 안정성 개선, 버그 수정 익명화되어 서버 전송

4. 데이터 저장 및 보안

🔒 보안 조치
  • 로컬 저장: 메모는 사용자의 기기에만 저장됩니다
  • 암호화: 민감한 데이터는 AES-256 암호화 적용
  • HTTPS: 모든 통신은 SSL/TLS 암호화
  • 접근 제한: 제3자 접근 차단
개인정보 보호: 사용자의 메모 내용은 서버로 전송되지 않으며, 오직 사용자의 기기에만 저장됩니다.
로컬 저장소 보호
브라우저 보안 정책에 따라 다른 웹사이트에서는 접근할 수 없습니다.

5. 사용자 권리

메모 앱 사용자는 다음과 같은 권리를 갖습니다:

열람권

저장된 모든 메모를 언제든지 열람할 수 있습니다.

수정권

메모 내용을 자유롭게 수정하거나 삭제할 수 있습니다.

이동권

메모를 텍스트 파일로 내보낼 수 있습니다.

삭제권

모든 메모를 완전히 삭제할 수 있습니다.

6. 데이터 보유기간

중요한 안내: 메모 앱은 사용자가 직접 삭제할 때까지 데이터를 보관합니다.
  • 메모 데이터: 사용자가 삭제할 때까지 기기에 보관
  • 기술적 로그: 수집 후 30일 내 자동 삭제
  • 브라우저 캐시: 브라우저 설정에 따라 관리
💡 데이터 관리 팁

정기적으로 불필요한 메모를 삭제하고, 중요한 메모는 별도로 백업하는 것을 권장합니다.
